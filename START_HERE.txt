📋 FIRST TIME SETUP (Do this ONCE):
═══════════════════════════════════════════════════════════

1. Go to: https://dashboard.ngrok.com/signup
   
2. Sign up (FREE - use Google/GitHub)

3. Copy your authtoken (long string of letters/numbers)

4. Open Command Prompt (search "cmd" in Windows)

5. Type: ngrok config add-authtoken YOUR_TOKEN
   (Replace YOUR_TOKEN with your actual token)

6. Press Enter

7. Done! You never need to do this again.


🎯 EVERY TIME YOU WANT TO SHARE:
═══════════════════════════════════════════════════════════

STEP 1: Double-click → 1_start_app.bat
        (A black window will open - KEEP IT OPEN)

STEP 2: Double-click → 2_start_ngrok.bat
        (Another window opens - KEEP IT OPEN)

STEP 3: In the ngrok window, you'll see:
        
        Forwarding: https://abc123.ngrok.io → localhost:5000
        
        Copy that URL (https://abc123.ngrok.io)

STEP 4: Send URL to friends via WhatsApp/Email

STEP 5: Friends open that URL in their browser - Done! 🎉


📱 FRIENDS CAN NOW:
═══════════════════════════════════════════════════════════

✓ Access from any device (phone, laptop, tablet)
✓ Access from anywhere in the world
✓ Design their own rooms
✓ Create accounts and save designs


🔄 WHEN YOU MAKE CHANGES TO CODE:
═══════════════════════════════════════════════════════════

1. Save your code files

2. In the app window (1_start_app.bat), press Ctrl+C

3. Press any key to close

4. Double-click 1_start_app.bat again

5. Tell friends to refresh their browser

6. No need to restart ngrok! Same URL works.


⚠️ IMPORTANT NOTES:
═══════════════════════════════════════════════════════════

• Keep BOTH windows open while friends are using it
• Your computer must stay on
• Free ngrok works for 8 hours continuously
• URL changes if you close and reopen ngrok
• If URL changes, send new URL to friends


🛑 TO STOP SHARING:
═══════════════════════════════════════════════════════════

1. Press Ctrl+C in both windows
2. Close both windows
3. Done!
